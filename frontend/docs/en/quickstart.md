# Quickstart

Five minutes to walk through the core flow.

## 1. Sign up

Open [cardnote.io](https://cardnote.io), click **Get started** in the top right. Sign up with email + verification code.

After signup the system seeds a few starter sets and one blank note as an example.

## 2. Write your first card

You'll land on the Notes entry. Click **+ New note** at the bottom and write today's spark.

> 💡 Titles are **optional** — just write; the first line shows as the title automatically. Use a Markdown `#` heading for a deliberate title; clip / file cards ship with a dedicated **Title** field.

## 3. Add fields

Open the note's `⋯` menu → **Note details**. The right panel of the full-page detail lets you add fields of nine types: `text` / `number` / `choice` / `date` / `checkbox` / `url` / `cardLink` / `cardLinks` / `collectionLink`. See [Properties](/docs/properties).

## 4. Organize across sets

Sidebar → **+ New set**. Drag cards in — a single card can **belong to multiple sets at once**: file it once by topic, again by time, etc. Position and pin state are independent in each set.

If you'd rather express "this card *relates to* a set without being a member", use the `collectionLink` property field.

## 5. Reference other cards

To link cards, **use a `cardLink` or `cardLinks` property** — e.g. add a "Book" field of type `cardLink` on a Reading note, target set "Topics/Books". The referenced card gets an automatic backlink at the bottom of its detail page.

The whole network is visible from the **Connections** entry.

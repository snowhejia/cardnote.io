# 卡片类型

cardnote 把所有信息抽象为**卡片**。下面 8 个内置类型与落地页的合集模板目录一一对应。

## 笔记 · 6 个子模板

| 子模板 | 用途 |
|---|---|
| **学习** | 课程笔记、教材摘要、复习日 |
| **读书笔记** | 章节摘抄 + 个人评论 |
| **视频笔记** | 含时间戳的视频要点 |
| **灵感** | 闪念、触发场景、下一步 |
| **日记** | 每日一卡，含心情、天气 |
| **摘抄** | 单纯的引文 |

## 文件 · 5 个子模板

| 子模板 | 内置字段 |
|---|---|
| **图片** | 拍摄时间、地点 |
| **视频** | 时长（自动）、分辨率（自动） |
| **音频** | 时长（自动）、表演者 / 播客 |
| **文档** | 页数、作者 |
| **其他** | 类型说明、备注 |

文件类型卡片支持上传后自动生成 WebP 缩略图，以及图片 / 视频 / 音频 / PDF 在卡片详情里直接预览。

## 主题 · 12 个子模板

| 子模板 |
|---|
| 人物、组织、地点、事件、概念、书籍 |
| 影视、动漫、音乐、游戏、课程、应用 |

每个子模板带有合理的内置字段，例如：
- 人物：身份、出生年、关联作品（cardLinks）
- 书籍：作者、出版社、ISBN、阅读状态
- 影视：导演、主演、上映年份、个人评分
- 应用：开发商、平台、官网

## 剪藏 · 14 个子模板

按平台拆开，每个剪藏模板有对应平台的元数据字段（原文链接、作者、发布时间等）。

| 子模板 |
|---|
| 网页、邮件、小红书、B 站、微信公众号 |
| 抖音、微博、知乎、豆瓣、GitHub |
| 推特 / X、Instagram、Reddit、App Store |

## 任务 · 3 个子模板

- **待办** · 普通 to-do，可设置提醒日期 / 时间
- **日程** · 带固定开始时间的安排
- **习惯** · 周期性勾选，含「连续天数」字段

## 项目 · 2 个子模板

- **在做** · 进行中的项目
- **已归档** · 已结束的项目

## 开支 · 3 个子模板

- **日常** · 一次性支出
- **订阅** · 周期性扣费（含下次扣费日期）
- **报销** · 待报销，附凭证

## 账户 · 3 个子模板

- **登录** · 平台账号（用户名、密码、登录链接）
- **银行卡** · 银行 / 卡号 / 有效期
- **证件** · 身份证 / 护照等

> ⚠️ **重要安全提示**：账户类卡片的密码 / 卡号字段当前以**明文**存储在你账号下，**不是**端到端加密。强敏感凭证（银行支付密码、CVV、身份证号）请自行评估后再放入。

> 注：部分子模板还在持续完善中，新建后可能没有完整的内置字段。可以先在合集设置 → 合集模板里手动加你需要的字段。

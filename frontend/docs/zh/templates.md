# 合集模板（Schema）

每个合集都可以有一份 **Schema**——定义这个合集里的卡片应该有哪些自定义字段。新建卡片时这些字段自动出现。

## Schema 包含什么

- **属性字段（schemaFields）** · 卡片可填的字段列表
- **autoLinkRules** · 保存时自动建立卡片关联的规则（见下文）

> 没有"正文模板"功能——cardnote 不预填正文。

## 在哪里编辑

合集右键 →「合集设置」→「合集模板」标签页。

UI 支持：
- 拖拽重排字段
- 改字段名、类型、默认值、颜色 / 图标
- 给 `choice` 类字段加 / 删 / 改选项
- 给 `cardLink` / `cardLinks` 字段指定目标合集

> 注意：**预设子模板（学习 / 读书笔记 / 网页剪藏 …）的 schema 是只读的**，避免预设升级时和你的修改冲突。要自定义，建议把卡片移到一个普通合集，再在那个合集上加自己的字段。

## autoLinkRules：自动建链

部分预设合集自带 `autoLinkRules`——保存卡片时根据某个字段值，自动在两张卡片之间建立 `cardLink`。

举个真实存在的例子：**视频笔记**子模板内置一条规则——如果你在「视频」字段里关联了一张视频文件卡片，保存后会自动在该视频卡片的"来源"字段里反向写入这张视频笔记，无需手动维护两端。

> 自动建链规则**不能**在 UI 里编辑，目前只能由预设携带。后续会开放给用户自定义。

## 无公式 / 计算字段

cardnote 当前**不支持**公式 / 计算字段。如果你需要从其它字段派生值，请在保存时手动填入。

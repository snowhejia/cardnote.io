# 快速开始

5 分钟跑通核心流程。

## 1. 注册账号

打开 [cardnote.io](https://cardnote.io)，点右上角「开始使用」。注册支持邮箱 + 验证码。

注册成功后系统会为你创建几个种子合集，并写入一张空白笔记当作示例。

## 2. 写第一张卡片

进入应用默认在「笔记」入口。在底部点「+ 新建笔记」，写下你今天的想法。

> 💡 标题是**可选**的——直接写正文，第一行会被自动当成标题展示；想要一个正经标题就用 Markdown `#`；剪藏 / 文件这类预设卡片自带「标题」字段。

## 3. 给卡片加字段

打开笔记的「⋯」菜单，点「笔记详情」。进入全页详情后，右侧会显示属性面板：你可以加 `text` / `number` / `choice` / `date` / `checkbox` / `url` / `cardLink` / `cardLinks` / `collectionLink` 这九种字段，详见 [属性](/docs/properties)。

## 4. 在合集里组织

侧栏点「+ 新建合集」，给个名字。把卡片拖进去——同一张卡片可以**同时属于多个合集**，按主题归一份、按时间再归一份都行。在每个合集里它的位置和置顶状态独立。

如果只想表达"这张卡片**和**某个合集相关、但不真的归属"，可以用 `collectionLink` 属性字段。

## 5. 引用其它卡片

要把卡片之间建立关联，**用 `cardLink` 或 `cardLinks` 属性字段**——比如读书笔记上加一个「书籍」字段，类型选 `cardLink`，目标合集设为「主题/书籍」。从此，被引用的「书籍」卡片底部会自动出现一个反向链接列表。

整个网络可以从「连接图」入口看到。
